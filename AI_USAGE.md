# AI Usage Disclosure

This document describes how AI tools were used in completing this assignment.

## Tools Used

- **Cursor** (AI-assisted editor)

## How AI Was Used

1. **Project setup**: AI helped create the project structure (Makefile, R scripts, directory layout).

2. **Code implementation**: AI assisted in writing `src/train.R` and `src/evaluate.R`, including:
   - Data loading and preprocessing in R
   - Feature selection using XGBoost importance
   - Hyperparameter tuning via 5-fold cross-validation
   - Model persistence with `saveRDS` and evaluation pipeline

3. **Documentation**: AI helped draft the README and this AI_USAGE.md file.

## Extent of Use

- AI was used as a coding assistant to accelerate implementation and structure the solution.
- All design choices (model type, feature selection, preprocessing, tuning strategy) were made with human oversight and rationale.
- The final model, methodology, and documentation were reviewed and refined by me.
